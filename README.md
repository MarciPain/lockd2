# Lockd2 Mobile App

A Flutter application for controlling remote locks. This app is designed to work with the **[lockd-go2](https://github.com/MarciPain/lockd-go2)** backend, which provides security, access control (ACL), and MQTT integration.

## Features
- **Remote Control**: Open or toggle locks from your phone.
- **ACL Support**: Only shows the locks you have permission to access (managed on the server).
- **Multi-language Support**: Fully localized in Hungarian and English (expandable).
- **Battery Status**: View the battery levels of your connected locks.

## Automatic Builds
This repository is configured with **GitHub Actions**. Every time you push code to the `main` branch, a new Android APK is automatically built.

You can find the compiled APKs in the **Actions** tab of the GitHub repository under the latest successful "Build Android" run.

## Setup
1. **Server Setup**: Ensure you have a running **[lockd-go2](https://github.com/MarciPain/lockd-go2)** instance.
2. **API Key**: Generate an API key on the server using `lockd2 -gen-key <username>`. 
3. **App Configuration**: 
   - Install [Flutter](https://docs.flutter.dev/get-started/install).
   - Run `flutter pub get`.
   - Open the app and enter the **Raw Key** generated by the server and your server's URL.
